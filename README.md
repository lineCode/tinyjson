tinyjson
========
Tiny (as in minimal) implementation of a JSON response parser. The parser
itself is only dependent on:
* `<string.h> - memcpy`
* `<stdlib.h> - atof`

Also included is a C++ wrapper around the low level `tinyjson_parse` function
